The overflow property has the following values:

  • visible, por defecto, El desbordamiento(overflow) no está recortado. El contenido se renderiza fuera de la caja del elemento.
  • hidden, el desbordamiento se recorta y el resto del contenido será invisible
  • scroll, se recorta el desbordamiento y se agrega una barra de desplazamiento para ver el resto del contenido
  • auto, similar al desplazamiento, pero agrega barras de desplazamiento solo cuando es necesario

Nota:La propiedad de overflow solo funciona para elementos de bloque con una altura especificada.

Nota: en OS X Lion (en Mac), las barras de desplazamiento están ocultas de forma predeterminada y solo se muestran cuando se utilizan (aunque "overflow: scroll" está configurado)

Overflow-x and Overflow-y

Las propiedades overflow-x y overflow-y especifican si cambiar el desbordamiento de contenido solo horizontal o verticalmente (o ambos):

  • overflow-x especifica qué hacer con los bordes izquierdo / derecho del contenido.
  • overflow-y especifica qué hacer con los bordes superior / inferior del contenido.

Clear & Float

La propiedad flotante de CSS especifica cómo debe flotar un elemento.

La propiedad clear de CSS especifica qué elementos pueden flotar junto al elemento despejado y en qué lado.

Propiedad Float

La propiedad float se utiliza para posicionar y formatear contenido

La propiedad float puede tener uno de los siguientes valores:

  • left: el elemento flota a la izquierda de su contenedor
  • right: el elemento flota a la derecha de su contenedor
  • none: el elemento no flota (se mostrará justo donde aparece en el texto). Esto es predeterminado
  • inherit: el elemento hereda el valor flotante de su padre

En su uso más simple, la propiedad float se puede usar para envolver texto alrededor de imágenes.

Clear and Clearfix

La propiedad clear, cuando usamos la propiedad float, y queremos el siguiente elemento debajo (no a la derecha ni a la izquierda), tendremos que usar la propiedad clear.

La propiedad clear especifica lo que debería suceder con el elemento que está al lado de un elemento flotante.

La propiedad clear puede tener uno de los siguientes valores:

  • none: el elemento no se coloca debajo de los elementos flotantes izquierdo o derecho. Esto es predeterminado
  • left: el elemento se empuja debajo de los elementos flotantes izquierdosr
  • right: el elemento se empuja debajo de los elementos flotantes derechos
  • both: el elemento se empuja debajo de los elementos flotantes izquierdo y derecho
  • inherit: el elemento hereda el valor clear de su padre
