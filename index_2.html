Esquema CSS

Contorno: es una línea trazada fuera del borde del elemento (la verde).
Es una línea que se dibuja alrededor de los elementos, FUERA de los bordes, para hacer que el elemento "se destaque".
¡El contorno difiere de las fronteras! A diferencia del borde, el contorno se dibuja fuera del borde del elemento y puede superponerse a otro contenido.
Además, el contorno NO forma parte de las dimensiones del elemento; el ancho y alto total del elemento no se ve afectado por el ancho del contorno.

With del contorno

La propiedad "outline-width" especifica el ancho del contorno y puede tener uno de los siguientes valores:

thin (típicamente 1px).

medium (típicamente 3px).

thick (típicamente 5px).

Un tamaño específicado en (px, em, pt, cm, etc)

Propiedad outline-color

Sirve para especificar el color de un contorno Estos pueden ser:

  • Nombre - especificas un color mediante un nombre, como "red"
  • HEX - specify a hex value, like "#ff0000"
  • RGB - specify a RGB value, like "rgb(255,0,0)"(rojo, verde, azul)
  • HSL - specify a HSL value, like "hsl(0, 100%, 50%)"

Propiedad outline-offset

La propiedad agrega espacio entre un contorno y el borde/borde de un elemento.
El espacio entre un elemento y su contorno es transparente.
