Colores

Podemos especificar el color de un texto. Estos pueden ser:

  • Nombre - especificas un color mediante un nombre, como "red"
  • HEX - specify a hex value, like "#ff0000"
  • RGB - specify a RGB value, like "rgb(255,0,0)"(rojo, verde, azul)
  • HSL - specify a HSL value, like "hsl(0, 100%, 50%)"

Fuentes

La fuente correcta tiene un gran impacto en la forma en que los lectores experimentan un sitio web.
También es importante elegir el color y el tamaño de texto correctos para la fuente.

Fuentes Genericas

En CSS hay cinco familias de fuentes genéricas:

  • Las fuentes "serif" tienen un pequeño trazo en los bordes de cada letra. Crean una sensación de formalidad y elegancia.
  • Las fuentes "Sans-Serif" tienen líneas limpias (sin trazos pequeños adjuntos). Crean un aspecto moderno y minimalista
  • "Monospace" todas las letras tienen el mismo ancho fijo. Crean un aspecto mecánico.
  • "Cursive" imitan la escritura humana.
  • "Fantasy " son fuentes decorativas / juguetonas.

Todas las tipografías (fuentes aplicados a textos) se reducen a una de estas familias

Propiedades de fuentes

En CSS, usamos la propiedad "font-family" para especificar la fuente de un texto

Es común que algunos navegadores no reconozcan las familia de fuente que usamos, y por eso es recomendable que la propiedad "font-family" deba contener varios nombres de fuente como un sistema de "reserva", para garantizar la máxima compatibilidad entre navegadores / sistemas operativos.
Comience con la fuente que desee y termine con una familia genérica (para permitir que el navegador elija una fuente similar en la familia genérica, si no hay otras fuentes disponibles).

Los nombres de las fuentes deben separarse con comas, y tendríamos:

This is a paragraph, shown in the Times New Roman font.

This is a paragraph, shown in the Arial font.

This is a paragraph, shown in the Lucida Console font.

Fuentes seguras

Las fuentes seguras para la web son fuentes que se instalan universalmente en todos los navegadores y dispositivos.

Sin embargo, no hay fuentes 100% completamente seguras para la web. Siempre existe la posibilidad de que una fuente no se encuentre o no esté instalada correctamente.
Por lo tanto, es muy importante usar siempre fuentes de reserva.

Esto significa que debe agregar una lista de "fuentes de copia de seguridad" similares en la propiedad. Si la primera fuente no funciona, el navegador intentará la siguiente, y la siguiente, y así sucesivamente.

Siempre termine la lista con un apellido de fuente genérico

Tamaño del texto

La propiedad "font-size" establece el tamaño del texto.
Sin embargo, no debe usar ajustes de tamaño de fuente para que los párrafos parezcan encabezados, o que los encabezados se vean como párrafos.
Siempre use las etiquetas HTML adecuadas, como "h1"- "h6" para encabezados y "p" para párrafos.

El valor del tamaño de fuente puede ser un tamaño absoluto o relativo.

Tamaño absoluto:
* Establece el texto en un tamaño especificado
* No permite a un usuario cambiar el tamaño del texto en todos los navegadores (malo por razones de accesibilidad)
* El tamaño absoluto es útil cuando se conoce el tamaño físico de la salida

Tamaño relativo:
* Establece el tamaño relativo a los elementos circundantes
* Permite a un usuario cambiar el tamaño del texto en los navegadores

Si no especifica un tamaño de fuente, el tamaño predeterminado para el texto normal, como los párrafos, es 16px (16px=1em).

Podemos establecer el tamaño de la fuete con pixeles, em, porcentajes o una combinación de porcentajes con em

Em vs PX

El texto por defecto tiene 16px, para calcularlo en em se usa la fórmula: píxeles/16=em

Los programadores suelen utilizar "em" antes de "px" ya que con el tamaño em, es posible ajustar el tamaño del texto en todos los navegadores.
Pero no siempre es así, en Internet Explorer por ejemplo, la letra crece más de lo debido cuando se hace un zoom.

Emparejamiento de fuentes

Tomamos varias reglas que hacen de nuestra tipografía exitosas en la web
Puedes revisar más información en reglas de emparejamiento

FONT

Puedes aplicar estilos de fuente a un texto de la manera tradicional:
body {
background-color: black;
font-family: Verdana, sans-serif;
font-size: 16px;
color: gray;
}

O puedes usar la propiedad "font":
.etiqueta {
font: 20px Arial, sans-serif;
}

Este segundo ejemplo ademas añade un espacio entre lineas:

.etiqueta {
font: italic bold 12px/30px Georgia, serif;
}

En este párrafo la fuente es italic y bold, el tamaño es 12 pizeles,
el interlineado es de 30 pixeles y la familia fuente es Georgia.
