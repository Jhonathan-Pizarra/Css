Enlaces

Con CSS los links pueden adquirir muchos estilos, podemos darles estilos en color, letra, fondo, forma...

Mundo al revés!

los enlaces se pueden diseñar de manera diferente dependiendo del "estado" en el que se encuentren.
Los cuatro estados de enlaces son:

  • a:link - un enlace normal y no visitado
  • a:visited - un enlace que el usuario ha visitado
  • a:hover - un enlace cuando el usuario pasa el ratón sobre él
  • a:active - un enlace en el momento en que se hace clic en él

Estado de enlace

Nota: "a:hover" DEBE ir después de "a:link" y "a:visited" en la cascada de CSS para que sea efectivo.

Nota: "a:active" DEBE ir después de "a:hover" en la definición de CSS para que sea efectivo.
