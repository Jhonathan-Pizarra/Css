Listas

En HTML habíamos visto que hay listas ordenadas "ol" y desordenadas "ul"
En Css las propiedades aplciadas a listas te permiten:

  1. Establecer diferentes marcadores de elementos de lista para listas ordenadas
  2. Establecer diferentes marcadores de elementos de lista para listas desordenadas
  3. Establecer una imagen como marcador de elemento de lista
  4. Agregar colores de fondo a listas y elementos de lista

Ejemplo de lsita desordenada:

  • Coffee
  • Tea
  • Coca Cola
  • Coffee
  • Tea
  • Coca Cola

Ejemplo de lista ordenada:

  1. Coffee
  2. Tea
  3. Coca Cola
  1. Coffee
  2. Tea
  3. Coca Cola
  • Coffee
  • Tea
  • Coca Cola

Posición

Puede ser fuera o dentro

Inside significa que las viñetas estarán dentro del elemento de la lista. Como es parte del elemento de lista, será parte del texto y empujará el texto al principio:

  • Coffee
  • Tea
  • Coca Cola

Outside significa que las viñetas estarán fuera del elemento de la lista. El inicio de cada línea de un elemento de lista se alineará verticalmente:

  • Coffee
  • Tea
  • Coca Cola
