La propiedad Display

La propiedad especifica si/como un elemento es mostrado
Cada elemento HTML tiene un valor de visualización predeterminado dependiendo del tipo de elemento que sea. El valor de visualización predeterminado para la mayoría de los elementos es .blockinline

Mostar en Bloque - display: block

siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende a la izquierda y a la derecha hasta donde puede).
Algunso elementos de tipo bloque son:

  • h1 - h6
  • p
  • fomrs
  • header
  • footer
  • section
  • div

Mostar en fila (linea) - display: inline

Un elemento en línea no comienza en una nueva línea y solo ocupa la anchura necesaria.
Algunso elementos de tipo en fila ( o linea) son:

  • span
  • a
  • img

Cambiar la manera

Como se mencionó, cada elemento tiene un valor de visualización predeterminado. Sin embargo, puede anular esto.
Cambiar un elemento en línea a un elemento de bloque, o viceversa, puede ser útil para hacer que la página se vea de una manera específica.

Establecer la propiedad de visualización de un elemento solo cambia la forma en que se muestra el elemento, NO el tipo de elemento que es.
Por lo tanto, un elemento en fila con un elemento de disposición "bloque" no puede tener otros elementos de bloque dentro de él.

A display property with a value of "block" results in a line break between the two elements.

No Mostrar - display: none

se usa comúnmente con JavaScript para ocultar y mostrar elementos sin eliminarlos y recrearlos.

Click to show panel

This panel contains a div element, which is hidden by default (display: none).

It is styled with CSS and we use JavaScript to show it (display: block).

How it works: Notice that the p element with class="flip" has an onclick attribute attached to it. When the user clicks on the p element, a function called myFunction() is executed, which changes the style of the div with id="panel" from display:none (hidden) to display:block (visible).
