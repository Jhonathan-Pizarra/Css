CSS Layout - width and max-width

width, max-width & margin: auto;

Un elemento a nivel de bloque siempre ocupa todo el ancho disponible (se extiende hacia la izquierda y la derecha hasta donde puede).

Establecer el ancho de un elemento a nivel de bloque evitará que se extienda hasta los bordes de su contenedor.

Luego, puede establecer los márgenes en automático, para centrar horizontalmente el elemento dentro de su contenedor.
El elemento ocupará el ancho especificado y el espacio restante se dividirá en partes iguales entre los dos márgenes:

El problema con esto ocurre cuando la ventana del navegador es más pequeña que el ancho del elemento. El navegador agrega una barra de desplazamiento horizontal a la página.

La solución, es usar 4l uso de max-width, en esta situación, mejorará el manejo del navegador de ventanas pequeñas. Esto es importante al hacer que un sitio se pueda utilizar en dispositivos pequeños

The position Property

La propiedad "position" especifica el tipo de método de posicionamiento utilizado para un elemento (estático, relativo, fijo, absoluto o pegajoso).

Hay cinco valores de posición diferentes:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Los elementos se colocan utilizando las propiedades top, bottom, left y right. Sin embargo, estas propiedades no funcionarán a menos que se establezca primero la propiedad de posición.
También funcionan de forma diferente según el valor de la posición.

position: static;

Los elementos HTML se colocan estáticos de forma predeterminada. Los elementos de posición estática no se ven afectados por las propiedades top, bottom, left y right.

This div element has position: static;

Un elemento con posición: estática; no se coloca de ninguna manera especial; está siempre posicionado de acuerdo con el flujo normal de la página:

position: relative;

Un elemento con posición: relativa; está posicionado en relación con su posición normal. Establecer las propiedades top, bottom, left y right de un elemento en una posición relativa hará que se ajuste lejos de su posición normal. El resto del contenido no se ajustará para encajar en ningún espacio dejado por el elemento.

This div element has position: relative;

Un elemento con posición: relativa; está posicionado en relación con su posición normal

position: fixed;

Un elemento con posición:fixed; se coloca en relación con la ventana gráfica, lo que significa que siempre permanece en el mismo lugar incluso si se desplaza la página. Las propiedades top, bottom, left y right. Se utilizan para colocar el elemento.

This div element has position: fixed;

Un elemento con posición: fijo; está posicionado en relación con la ventana gráfica, lo que significa que siempre permanece en el mismo lugar incluso si se desplaza por la página:

position: absolute;

Un elemento con posición: absoluta; se posiciona en relación con el antepasado posicionado más cercano (en lugar de posicionado en relación con la ventana gráfica, como fijo).

Sin embargo; si un elemento de posición absoluta no tiene antepasados de posición, utiliza el cuerpo del documento y se mueve junto con el desplazamiento de la página.

Los elementos de posición absoluta se eliminan del flujo normal y pueden superponerse a elementos.

Un elemento con posición: absoluta; está posicionado en relación con el antepasado posicionado más cercano (en lugar de posicionado en relación con la ventana gráfica, como fijo)

position: sticky;

Un elemento con posición: pegajoso; se posiciona según la posición de desplazamiento del usuario.

Un elemento adhesivo alterna entre relativo y fijo, según la posición de desplazamiento. Se coloca en una posición relativa hasta que se alcanza una posición de desplazamiento determinada en la ventana gráfica, luego se "pega" en su lugar (como la posición: fija).

Internet Explorer no admite el posicionamiento fijo. Safari requiere un prefijo -webkit- (lo veremos a continuación). También debe especificar al menos una de las opciones superior, derecha, inferior o izquierda para que funcione la posición adhesiva.

Elementos superpuestos

Cuando los elementos se colocan, pueden superponerse a otros elementos. La propiedad z-index especifica el orden de pila de un elemento (qué elemento debe colocarse delante o detrás de los demás).

Un elemento puede tener un orden de pila positivo o negativo:

This is a heading

Debido a que la imagen tiene un índice Z de -1, se colocará detrás del texto.

Un elemento con un orden de pila mayor siempre está delante de un elemento con un orden de pila menor.

Si dos elementos se superponen sin un índice z especificado, el elemento colocado en último lugar en el código HTML se mostrará en la parte superior.
